# AQI App - User Manual

## Welcome to Your Personal AQI Assistant!

This app uses advanced AI agents to provide you with personalized air quality insights, predictions, and recommendations.

## Getting Started

### 1. First Launch
When you first open the app, you'll see the home screen displaying:
- Current AQI level
- Weather conditions
- AI-generated insights
- Personalized recommendations

### 2. Understanding the Interface

#### Home Screen
- **AQI Circle**: Shows current air quality index with color coding
  - Green (0-50): Good
  - Yellow (51-100): Moderate
  - Orange (101-150): Unhealthy for Sensitive Groups
  - Red (151-200): Unhealthy
  - Purple (201-300): Very Unhealthy
  - Maroon (301+): Hazardous

- **AI Insights Card**: Real-time analysis of current air quality
- **Recommendations Card**: Personalized advice based on your profile
- **Weather Info**: Current temperature, humidity, and conditions
- **7-Day Forecast**: Upcoming weather and AQI trends

#### Chat Screen
Access the AI assistant by tapping the chat icon in the navigation bar.

**What You Can Ask:**
- "What's the current AQI?"
- "Should I go for a run?"
- "What will the air quality be tomorrow?"
- "What if traffic increases by 50%?"
- "Is it safe to exercise outdoors?"

**How It Works:**
The AI assistant uses multiple specialized agents to:
1. Fetch real-time data
2. Analyze air quality patterns
3. Generate predictions
4. Provide personalized recommendations

#### Habits Screen
Track your daily routines and exposure patterns:
- Record outdoor activities
- Log commute times
- Track health symptoms
- View historical data

#### Prediction Screen
View detailed forecasts:
- 24-hour AQI predictions
- Hourly breakdowns
- Confidence scores
- Trend analysis

#### Route Exposure (New!)
Calculate PM exposure on your commute:
1. Enter your route
2. View exposure analysis
3. Get route score (0-100)
4. Receive optimization suggestions

## Customizing Your Experience

### Setting Up Your Profile
1. Go to Habits screen
2. Tap "Edit Profile"
3. Enter your information:
   - Age
   - Location
   - Health conditions (respiratory issues, heart condition, pregnancy)
   - Daily routes
   - Activity preferences

### Managing Notifications
1. Go to Settings
2. Select "Notifications"
3. Choose:
   - Alert threshold (AQI level)
   - Frequency (hourly, daily)
   - Quiet hours

### Saving Favorite Locations
1. Tap location name on home screen
2. Select "Add to Favorites"
3. Name your location
4. Access quickly from dropdown

## Troubleshooting

### App Not Updating
- Check internet connection
- Pull down to refresh
- Restart app

### No Predictions Available
- Requires historical data
- Wait 24 hours after first use
- Check location permissions

### Chat Not Responding
- Verify API key is set
- Check internet connection
- Try simpler queries

### Inaccurate Recommendations
- Update your profile
- Verify location is correct
- Report issue via feedback

## Privacy & Data

### What We Collect
- Location (for AQI data)
- User profile (stored locally)
- Query history (last 100)
- Habit data

### What We Don't Collect
- Personal identification
- Precise GPS tracking
- Third-party sharing
- Advertising data

### Data Storage
- All data stored locally on your device
- No cloud sync (optional in future)
- Delete anytime via Settings

## Glossary

- **AQI**: Air Quality Index (0-500 scale)
- **PM2.5**: Fine particulate matter (< 2.5 micrometers)
- **PM10**: Coarse particulate matter (< 10 micrometers)
- **µg/m³**: Micrograms per cubic meter
- **Confidence Score**: Prediction reliability (0-100%)
